# 顶部固定菜单栏和侧边悬浮按钮组优化

## 需求分析
1. **顶部固定菜单栏**：
   - 左侧显示"计算机科学技术学院"
   - 右侧横向排列功能区：带图标的登录/注册按钮、我的部门、在线客服、加入
   - 与轮播图隔开，不是融为一体
   - 当用户浏览到顶部以下内容时，收起隐藏，只保留左侧标志和右侧用户图标

2. **侧边悬浮按钮组**：
   - 在线客服按钮
   - 加入按钮
   - 一键置顶按钮

## 实现步骤

### 1. 修改Navbar组件
- **调整布局**：确保菜单栏与轮播图隔开，不是融为一体
- **功能区排列**：右侧横向排列：登录/注册按钮、我的部门、在线客服、加入
- **滚动监听**：添加滚动监听，实现菜单栏的收起和展开
- **响应式设计**：确保在不同屏幕尺寸下都能正常显示

### 2. 实现菜单栏收起隐藏功能
- **滚动判断**：当用户浏览到顶部以下内容时，收起隐藏
- **保留元素**：只保留左侧"计算机科学技术学院"标志和右侧用户图标
- **动画效果**：添加平滑的收起和展开动画

### 3. 修改侧边悬浮按钮组
- **添加加入按钮**：在侧边悬浮按钮组中添加加入按钮
- **调整顺序**：确保按钮顺序合理
- **样式统一**：保持与其他按钮一致的样式

### 4. 测试和优化
- **功能测试**：确保菜单栏的收起和展开功能正常
- **样式测试**：确保与轮播图隔开，不是融为一体
- **响应式测试**：确保在不同屏幕尺寸下都能正常显示
- **性能优化**：确保滚动监听不会影响页面性能

## 预期效果
1. **顶部固定菜单栏**：
   - 与轮播图隔开，不是融为一体
   - 右侧横向排列功能区：登录/注册按钮、我的部门、在线客服、加入
   - 当用户浏览到顶部以下内容时，收起隐藏，只保留左侧标志和右侧用户图标

2. **侧边悬浮按钮组**：
   - 包含在线客服按钮、加入按钮、一键置顶按钮
   - 样式统一，位置合理
   - 方便用户快速访问

## 技术栈
- React
- Framer Motion（动画效果）
- Tailwind CSS（样式）
- Next.js（框架）
